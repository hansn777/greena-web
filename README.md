# GreenA Website

도메인별 퍼블리싱을 위한 사이트별 폴더 구조입니다.

## 사이트 구조

### 📁 main/

메인 사이트 (홈페이지)

- 모든 히어로 슬라이드 이미지 포함
- 플로팅 액션 버튼 이미지 포함
- 제품 슬라이더 이미지 포함

### 📁 company/

회사소개 사이트

- 회사소개 관련 페이지들
- 로고 이미지만 포함

### 📁 map/

지도 사이트

- 지도 페이지
- 로고 이미지만 포함

### 📁 test/

테스트 사이트

- 테스트 페이지
- 로고 이미지만 포함

## 이미지 파일 그룹핑

각 사이트 폴더에는 해당 사이트에서 사용되는 이미지 파일들만 포함되어 있습니다:

- **main/**: 모든 이미지 파일 (로고, 배경, 히어로 슬라이드, 플로팅 버튼)
- **company/**: 로고 이미지만
- **map/**: 로고 이미지만
- **test/**: 로고 이미지만

## 배포 방법

각 폴더를 독립적인 도메인 또는 서브도메인으로 배포할 수 있습니다.
