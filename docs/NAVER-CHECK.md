# ✅ 네이버 제출 확인 가이드

## 📋 제출 상태 확인 방법

### 1. 네이버 서치어드바이저 접속
- https://searchadvisor.naver.com/ 접속
- 네이버 계정으로 로그인

### 2. 사이트맵 제출 확인

1. **등록한 사이트 선택**
   - 웹마스터 도구에서 `greenaofficial.com` 선택

2. **사이트맵 제출 현황 확인**
   - 왼쪽 메뉴에서 **"요청"** → **"사이트맵"** 클릭
   - 또는 **"검색엔진 최적화"** → **"사이트맵"** 클릭

3. **확인할 내용**
   - ✅ **제출된 사이트맵 목록**에 `https://greenaofficial.com/sitemap.xml`이 표시되는가?
   - ✅ **상태**가 "처리 중" 또는 "완료"로 표시되는가?
   - ✅ **제출 일시**가 표시되는가?
   - ✅ **URL 수**가 표시되는가? (예: 6개)

4. **정상 제출 확인 기준**
   - 사이트맵 URL이 목록에 표시됨
   - 상태가 "처리 중" 또는 "완료"
   - 오류 메시지가 없음

### 3. 수집 요청 확인

1. **수집 요청 현황 확인**
   - 왼쪽 메뉴에서 **"요청"** → **"수집 요청"** 클릭
   - 또는 **"검색엔진 최적화"** → **"수집 요청"** 클릭

2. **확인할 내용**
   - ✅ **요청한 URL 목록**에 `https://greenaofficial.com/`이 표시되는가?
   - ✅ **상태**가 "대기 중", "처리 중", 또는 "완료"로 표시되는가?
   - ✅ **요청 일시**가 표시되는가?

3. **정상 제출 확인 기준**
   - URL이 목록에 표시됨
   - 상태가 "대기 중" 이상
   - 오류 메시지가 없음

### 4. 수집 현황 확인 (가장 중요!)

1. **수집 현황 확인**
   - 왼쪽 메뉴에서 **"수집 현황"** 클릭
   - 또는 **"검색엔진 최적화"** → **"수집 현황"** 클릭

2. **확인할 내용**
   - ✅ **수집된 페이지 수**가 표시되는가?
   - ✅ **최근 수집 일시**가 표시되는가?
   - ✅ **수집 오류**가 있는가?

3. **정상 작동 확인 기준**
   - 수집된 페이지 수가 0보다 큼 (예: 6개)
   - 최근 수집 일시가 표시됨
   - 오류가 없음

### 5. 색인 현황 확인

1. **색인 현황 확인**
   - 왼쪽 메뉴에서 **"색인 현황"** 클릭
   - 또는 **"검색엔진 최적화"** → **"색인 현황"** 클릭

2. **확인할 내용**
   - ✅ **색인된 페이지 수**가 표시되는가?
   - ✅ **색인 대기 중인 페이지 수**가 표시되는가?

3. **정상 작동 확인 기준**
   - 색인된 페이지 수가 점진적으로 증가
   - 색인 대기 중인 페이지가 있으면 정상 (처리 중)

- [ ] 사이트맵 상태가 "처리 중" 또는 "완료"인가?
- [ ] 수집 요청 목록에 메인 페이지 URL이 표시되는가?
- [ ] 수집 요청 상태가 "대기 중" 이상인가?

### 1-2일 후 확인
- [ ] 수집 현황에 페이지 수가 표시되는가?
- [ ] 최근 수집 일시가 업데이트되었는가?
- [ ] 수집 오류가 없는가?

### 1-2주 후 확인
- [ ] 색인 현황에 색인된 페이지 수가 증가했는가?
- [ ] 네이버에서 `site:greenaofficial.com` 검색 시 결과가 나오는가?

## ⚠️ 문제가 있는 경우

### 사이트맵이 목록에 없음
**원인**: 제출이 실패했거나 아직 처리되지 않음
**해결**: 
1. 사이트맵 URL이 올바른지 확인: `https://greenaofficial.com/sitemap.xml`
2. 브라우저에서 직접 접속하여 XML 파일이 보이는지 확인
3. 다시 제출 시도

### 사이트맵 상태가 "오류"
**원인**: 사이트맵 형식 오류 또는 접근 불가
**해결**:
1. `sitemap.xml` 파일이 `public_html` 루트에 있는지 확인
2. 파일이 올바른 XML 형식인지 확인
3. 파일 권한 확인 (644)

### 수집 현황에 페이지가 0개
**원인**: 아직 크롤링이 시작되지 않음
**해결**:
1. 1-2일 정도 기다리기 (처리 시간 필요)
2. 수집 요청을 다시 제출
3. `robots.txt` 파일이 크롤링을 막지 않는지 확인

### 수집 오류가 있음
**원인**: 페이지 접근 불가 또는 오류
**해결**:
1. 오류 메시지 확인
2. 해당 페이지가 실제로 접근 가능한지 확인
3. `robots.txt`에서 차단되지 않았는지 확인

## 🔗 직접 확인 방법

### 1. 사이트맵 접속 확인
브라우저에서 직접 접속:
- `https://greenaofficial.com/sitemap.xml`
- XML 형식의 파일이 보여야 함

### 2. robots.txt 접속 확인
브라우저에서 직접 접속:
- `https://greenaofficial.com/robots.txt`
- 텍스트 파일이 보여야 함

### 3. 네이버 검색 테스트
네이버 검색창에서:
- `site:greenaofficial.com` 검색
- 사이트가 인덱싱되었다면 결과가 나타남

## 💡 팁

1. **처리 시간**: 사이트맵 제출 후 처리되는데 몇 시간에서 하루 정도 걸릴 수 있습니다.
2. **정기 확인**: 매일 또는 며칠에 한 번씩 수집 현황을 확인하세요.
3. **인내심**: 색인에는 1-2주 정도 걸릴 수 있습니다.
4. **오류 모니터링**: 수집 오류가 있으면 즉시 확인하고 해결하세요.
