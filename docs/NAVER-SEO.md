# 🔍 네이버 검색 노출 가이드

## 네이버 검색에 사이트가 나타나지 않는 이유

1. **네이버에 사이트가 등록되지 않음**
2. **사이트맵이 없음**
3. **네이버가 사이트를 크롤링하지 않음**
4. **사이트가 최근에 배포되어 아직 인덱싱되지 않음**

## ✅ 해결 방법

### 1단계: 네이버 서치어드바이저에 사이트 등록

1. **네이버 서치어드바이저 접속**
   - https://searchadvisor.naver.com/ 접속
   - 네이버 계정으로 로그인

2. **사이트 등록**
   - "웹마스터 도구" → "사이트 추가"
   - 사이트 URL 입력: `https://greenaofficial.com`
   - 사이트 소유 확인 방법 선택:
     - **방법 1: HTML 파일 업로드** (권장)
       - 네이버에서 제공하는 HTML 파일을 다운로드
       - `public_html` 폴더에 업로드
       - 네이버에서 확인 버튼 클릭
     - **방법 2: 메타 태그 추가**
       - 네이버에서 제공하는 메타 태그를 복사
       - `index.html`의 `<head>` 섹션에 추가

3. **사이트 소유 확인 완료**

### 2단계: 사이트맵 제출

1. **사이트맵 생성 확인**
   - `sitemap.xml` 파일이 `public_html` 루트에 있는지 확인
   - URL: `https://greenaofficial.com/sitemap.xml`

2. **네이버 웹마스터도구에서 사이트맵 제출**
   - 등록한 사이트 선택
   - "요청" → "사이트맵 제출"
   - 사이트맵 URL 입력: `https://greenaofficial.com/sitemap.xml`
   - 제출 버튼 클릭

### 3단계: robots.txt 확인

1. **robots.txt 파일 확인**
   - `robots.txt` 파일이 `public_html` 루트에 있는지 확인
   - URL: `https://greenaofficial.com/robots.txt`

2. **네이버가 크롤링할 수 있도록 설정 확인**
   - `User-agent: Yeti` (네이버 봇) 허용 확인
   - `Allow: /` 설정 확인

### 4단계: 수집 요청

1. **네이버 웹마스터도구에서 수집 요청**
   - 등록한 사이트 선택
   - "요청" → "수집 요청"
   - 메인 페이지 URL 입력: `https://greenaofficial.com/`
   - 요청 버튼 클릭

### 5단계: 대기 및 확인

1. **인덱싱 대기**
   - 네이버가 사이트를 크롤링하고 인덱싱하는데 **1-2주** 정도 소요될 수 있습니다
   - 최대 **1개월**까지 걸릴 수 있습니다

2. **진행 상황 확인**
   - 네이버 웹마스터도구 → "수집 현황"에서 확인
   - "색인 현황"에서 인덱싱된 페이지 수 확인

3. **검색 테스트**
   - 네이버에서 `site:greenaofficial.com` 검색
   - 또는 `그린에이` 검색

## 📋 체크리스트

배포 전:
- [ ] `sitemap.xml` 파일이 `public_html` 루트에 있는가?
- [ ] `robots.txt` 파일이 `public_html` 루트에 있는가?
- [ ] 모든 페이지에 적절한 메타 태그가 있는가?

배포 후:
- [ ] 네이버 서치어드바이저에 사이트 등록 완료
- [ ] 사이트맵 제출 완료
- [ ] 수집 요청 완료
- [ ] `https://greenaofficial.com/sitemap.xml` 접속 가능한가?
- [ ] `https://greenaofficial.com/robots.txt` 접속 가능한가?

## 🔍 추가 SEO 최적화

### 네이버 검색 최적화

1. **네이버 지식백과 등록** (선택사항)
   - 회사 정보를 네이버 지식백과에 등록하면 검색 노출이 더 좋아집니다

2. **네이버 블로그 활용**
   - 네이버 블로그에 회사 소개 및 제품 정보 게시
   - 본사 사이트로 링크 연결

3. **네이버 비즈니스 등록** (선택사항)
   - 네이버 플레이스에 회사 정보 등록

### 키워드 최적화

현재 메인 페이지에 포함된 키워드:
- 공기정화장치
- 공기청정기
- 렌탈
- 유지관리
- 그린에이

추가로 고려할 키워드:
- 공기청정기 렌탈
- 공기정화장치 전문기업
- 공기청정기 유지관리
- 공기청정기 필터 교체

## ⚠️ 주의사항

1. **인덱싱은 시간이 걸립니다**
   - 즉시 검색 결과에 나타나지 않을 수 있습니다
   - 최소 1-2주, 최대 1개월까지 걸릴 수 있습니다

2. **정기적인 업데이트**
   - 사이트맵을 정기적으로 업데이트하세요
   - 새로운 페이지가 추가되면 사이트맵에 반영하세요

3. **네이버 웹마스터도구 모니터링**
   - 정기적으로 웹마스터도구를 확인하여 크롤링 오류가 없는지 확인하세요
